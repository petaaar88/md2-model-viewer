DESCRIPTION

   This is a quake2 player model that allows you to play Quake2 
   as a Martian from the film Mars Attacks.  
   See PREVIEW.JPG


INSTALLATION

   Copy the directory called MARTIAN and its content to your 
   quake2\baseq2\players\
   directory.

TO USE

   Run quake2 go to Multiplay / PlayerSetup / Model and select
   MARTIAN. You can then choose a skin.

   If that doesn't work, or you get the "No Valid Players Found"
   bug, just add the following to your cfg file at the bottom.
	
	skin martian/martian
